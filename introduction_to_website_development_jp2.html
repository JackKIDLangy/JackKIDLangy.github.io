    HTMLの文法の初歩

    それでは基本的なHTMLのコードの具体例を示し、解説します。 
                    
                    <!DOCTYPE html>
                        <html>
                            <head>
                                <title>ここにタイトル</title>
                            </head>
                            <body>
                                <h1>最初のWebページ例</h1>
                                 <p>何かしらの文章</p>
                                <ul>
                                    <li>列挙したいもの1</li>
                                    <li>列挙したいもの2</li>
                                    <li>列挙したいもの3</li>
                                >/ul>
                             </body>
                         </html>
    実際にHTMLのコードを見て気づいた人は沢山いるでしょうが、不等号がコードで多く書かれていますね。
    そして不等号で始めて単語を書いて不等号で閉じるということをしています。
    そしてその不等号はすべて半角です。
    この不等号こそがHTMLの本質であり、第1講で出てきたタグという概念です。そう、これをタグと呼ぶのです。
    タグは不等号の次にスラッシュ(/)を入れたタグとペアで使われます。 そのひとかたまりを要素と呼びます。
    例えば先ほど示したコード内にある、
            
            <li>列挙したいもの1>/li>
    が1つの要素となっています。これをli要素と呼びます。同様に、 
            
            <ul>
                <li>列挙したいもの1>/li>
                <li>列挙したいもの2>/li>
                <li>列挙したいもの3>/li>
            </ul>
    をul要素と呼びます。新しい用語がどんどん出てきてつらいかもしれませんが、まだ続きますよ。例えば、 
            
            <li>列挙したいもの1>/li>
    における
    <li>
    を開始タグと呼びます。そしてスラッシュ(/)をつけている
    </li>
    を終了タグと呼びます。
    つまり、まとめると要素は開始タグと終了タグのペアで構成されています。
    そしてそれぞれの要素は不等号で囲まれていて、開始タグは
    <使いたいタグの名前>
    で終了タグは
    </使いたいタグの名前>
    です。
    終了タグのスラッシュ(/)を忘れないようにしてください。
    まあ、僕はVisual Studio Codeを使っていますが、開始タグを書いたら自動的に終了タグが書かれるため忘れることはありませんが。
    なので、Visual Studio Codeはおすすめです。
    そしてそのようなエディタの機能なしでタグを実際に書くとするならば、以下の手順になります。
    1. まずタグの中身つまりタグで囲みたいものの前と後ろに
    <>
    をつける。
    2. 開始タグの中身つまり使いたいタグ名を書く
    3. 終了タグを /使いたいタグ名 と書く、(くどいですがスラッシュ(/)を忘れずに！
    といった感じでタグの一般的な使い方がわかったと思います。

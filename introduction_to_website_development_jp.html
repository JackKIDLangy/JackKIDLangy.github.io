そもそもインターネットとは何か

インターネットとはTCP/IPなどの標準化されたプロトコルを用いて相互に通信するコンピュータやサーバーのグローバルネットワークであり ウェブサイト、電子メール、その他のデジタルなコミュニケーションを通じて、テキスト、画像、動画、音声などの情報やリソースを共有することができます。
インターネットの主要なアプリケーションであるワールドワイドウェブ（WWWまたはWeb）は、Webブラウザを介してアクセス可能な、相互にリンクされたハイパーテキストの文書のシステムです。
TCP/IPとは、Transmission Control Protocol/Internet Protocolの略称です。
インターネットを含むネットワーク上の機器間の通信に使用されるプロトコルのセットです。TCPプロトコルは、機器間の信頼性の高い接続を確立し、データが正しくパケットに分割され、正しい順序で送信および受信されるようにする役割を果たします。IPプロトコルは、データのパケットを適切な宛先にルーティングする役割を担っています。これらのプロトコルを組み合わせることで、物理的な場所やネットワークアーキテクチャに関係なく、機器間の通信に信頼性と効率性を持たせることができます。
簡単に言うと、TCP/IPは、インターネット上でコンピュータ同士が会話し、情報を共有するための通信言語です。インターネットのバックボーンであり、インターネット上でデータを伝送・配信する方法の基礎であり、今日私たちが利用しているすべてのインターネットサービスやアプリケーションを可能にするものです。
プロトコルは、機器やシステムが互いにどのように通信すべきかを規定する一連の規則と標準のことです。データを送信し、機器やシステムが送信された情報を理解し応答できるようにするための標準化された方法です。プロトコルは、コンピューターネットワーキング、テレコミュニケーション、輸送など、さまざまな分野や産業で使用されています。
コンピュータネットワークの分野では、コンピュータ、サーバー、ルーターなど、ネットワーク上のさまざまな機器やシステム間の通信を制御するためにプロトコルが使用されています。具体例を挙げると、インターネット上の機器間の通信には先ほど出てきたTCP/IPプロトコルが、WebブラウザーとWebサーバー間の通信にはあなたは一度は見たことがあるでしょうが、HTTPプロトコルが使用されています。プロトコルがなければ、異なる機器やシステムが互いに効果的かつ効率的に通信することができません。

GitHub Pagesで手っ取り早く静的なWebサイトを作ってみる

そもそもGitHubって何?

GitHubは、ユーザーがソフトウェア開発プロジェクトを保存・管理するためのウェブベースのプラットフォームです。GitHubは、ソフトウェア開発者がコラボレーション、コードの共有とレビュー、問題やバグの追跡、コードの異なるバージョンの管理と保守を行うための一元的な場所を提供します。Gitバージョン管理システム上に構築されており、開発者は自分のコードの変更点を追跡し、他のチームメンバーと共同作業することができます。さらに、GitHubは、プルリクエスト、プロジェクト管理ツール、他のツールやサービスとの統合など、幅広い機能を提供しています。世界中の開発者や組織で、ソフトウェアプロジェクトのホスト、共有、コラボレーションに広く利用されており、最も人気のある開発者プラットフォームの1つとなっています。

じゃあそもそもGitHub Pagesって何?

GitHub Pagesは、GitHubのリポジトリから直接Webサイトを簡単に作成・ホストできるGitHubの機能です。例えば僕のこのサイトはGitHub Pagesの一例です。開発者は静的なウェブサイトやウェブアプリケーションを作成し、GitHubが提供するサブドメインやカスタムドメインにホストすることができます。GitHubのバージョン管理システムを利用することで、ユーザーは簡単にウェブサイトの更新や変更を行うことができ、異なるバージョンのウェブサイトを維持することができます。GitHub Pagesは、Jekyll、Hexo、Middlemanといった様々な静的ウェブサイトジェネレーターをサポートしており、シンプルなHTMLやCSSのウェブサイトをホストするのにも利用できます。さらに、GitHub PagesはHTTPSもサポートしており、カスタムドメインを使用するように設定することができるため、開発者が個人およびプロジェクトのウェブサイトをホストするための便利で簡単な方法となっています。

実際にWebサイトを作る前に

これからはみなさんは主にHTML5とCSS3という言語を使って実際にWebサイトを作っていきます。HTML文書はChromeなどのブラウザを使えばWebページとして表示されます。そしてテキストエディタを使えばテキスト文書、つまりコードとしても見ることができるのです。
先ほど最初の方でみなさんにインターネットやWebサイトなどについて詳しく説明しましたが、ここではもっと簡単に解説することにしましょう。HTMLで書かれたWebページを見るためには自分のコンピュータをインターネットに接続させて、アドレスを入力します。
それはサーバーというコンピュータがまず存在し、そしてそのコンピュータとインターネットを用いて繋がり、そこからHTML文書を引き出すことに成功してできていることなのです。
サーバーというのは、別にみなさんのコンピュータでも原理的には良いのですが、大抵はどこかの企業の大型サーバーの一部を何らかの契約をしてそこにHTML文書を渡してWebページを作っています。そのようなサーバーを運営している会社は、みなさんにサーバーを貸し出して24時間誰もがWebページを見ることができるようするかわりに料金を得て事業として成り立っています。
そして、実用上みなさんはこれから、ただHTML文書だけをサーバーに預ければ良いわけではなく画像データやCSSファイルなど色々なものを作成して渡さなければ、良い機能を持ったWebページを作ることができないのです。そして、それらの様々なみなさんが作成したコードやデータをひとまとまりにして、1つのフォルダに保存して、サーバーに送るのです。そのひとまとまりにしたフォルダをサイトと呼びます。
このサイトは作成者のコンピュータにバックアップをとっておき、何らかの編集をして更新をすればその都度サーバーに更新したファイルを送信します。

Webサイトを作るにあたって必要なもの

それはインターネットに繋がったみなさんがおそらくほぼ全員持っているであろうコンピュータとテキストエディタです。テキストエディタとは何かというと、使用者がテキストファイルを作成し、編集や保存をすることを可能にするコンピュータプログラムの一種です。
テキストエディタは、僕(横田健太)がみなさんにこのサイト上で解説するために例えばVisual Studio Codeというテキストエディタを使うように、学習者のみならずプログラマや他の何らかの専門家やありとあらゆるプログラミングという行為をする人たちによく使われるものなのです。 テキストエディタの利点を説明しましょう。まず一般的にテキストエディタは機能としてテキストを扱う作業、つまり編集作業の効率化を図るために構文の強調をしたり、コードを書く際に何らかの文字を打ち込んだら予期されるコードを提案してくれたり、コードの文法的な間違いを指摘してくれたするとても実用上便利なところがあります。

HTMLとは何か

HTMLとはHypertext Markup Languageの略称です。これだけでは何かわかりませんね、説明します。HTMLはWebサイトの構造とレイアウトを作成するためによく使われる標準的なマークアップ言語です。マークアップ言語とは、文書の構造を記述するために使用される言語です。HTMLの他にもXMLなどが代表的なマークアップ言語です。マークアップ言語を使うことにより、文書にタイトルや段落、リンクなどの構造を記述することができます。これでブラウザなどのアプリケーションはそれらの構造を理解して適切な形で表示することができます。
HTMLは例えばヘッダーやリスト、画像、リンクなど色々な文書やその他のものを定義するためにタグや属性があります。WebブラウザはHTML文書を受信するとコードを読み込んで、見出しを大きな文字で表示したり、リンクに下線を引いたり、画像を埋め込んだりしてコンテンツを表示しますが、それを適切に意図通りに扱うためにタグなどの勉強が必要となります。また、レスポンスデザインという概念があり、パソコン以外でも例えばスマートフォンやiPadなどのタブレット端末など異なるデバイスにそれぞれ異なる適した表示方法を指定することも可能なのです。それは実用上とても大切です。

CSSとは何か

CSS（Cascading Style Sheets）は、マークアップ言語(例えば先ほど説明したHTML)で書かれた文書の体裁を記述するために用いられるスタイルシート言語である。 スタイルシート言語とはコンピュータ言語の一種であり、先ほどの説明と重複しますが、文書の表示や書式を記述するために使われます。 そもそもスタイルシートは文書のデザインをその構造や内容から分離して考えて作っていくために使われるものであり、ＨＴＭＬなどの基本的なコードの構造を変更することなくデザインの作成や維持、更新を簡単に行うことができるものです。 話を元に戻すと、ＣＳＳは主にHTMLやXMLで書かれたWebページにスタイルやレイアウトを追加するために使用されるが、他の種類の文書にも適用できます。CSSでは、セレクタを使用して文書内の特定の要素を対象とし、色、フォント、レイアウトなどのスタイルを適用することができます。 また、CSSは、さまざまな画面サイズに適応する先ほど説明したレスポンシブデザインや、アニメーションなどの効果を作成する機能など、幅広いレイアウトおよびデザイン機能を提供します。CSSは、視覚的に豊かで魅力的なウェブサイトを作成するための強力なツールです。

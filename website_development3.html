CSSに触れてみる

CSSってそもそもなんだっけ?

CSSとはめっちゃ簡単に言うとWebページのデザインをコントロールするようなことができる言語でした。
HTMLにそのまま正しく記述すればそりゃ正しく表現できます。でもそれってなんだか2000年代初頭な感じがしませんか? いやまあ僕今日2023年1月18日では22歳でそんな時代の記憶があまりなくてお前がそんなこと言えるのかよって気もしますね。
まあそんなことはどうでもよくて、ただの所謂初期設定の状態から、ユーザーであるあなた側で何かしらサイトのレイアウト、 例えば文字や背景などの色や大きさなどを変えたりしていこうっていうのがこの第3講です。
でもそれを意図通りにするにはまずCSSの文法を理解しなければなりません。そんなことを僕とやっていきましょう!
とりあえずまずはCSSのファイルを作成してから文法に入門しましょう。
僕はVSCodeを使っているのですが、ここでintroduction_to_website_development_jp3.cssというCSSファイルを作ってみます。
今実際に実況するわけです。みなさんが見る頃には何か第2講のページとはレイアウトなどが少し違うなと思うことでしょう。
ということで僕は今、まさに今そのCSSファイルを作成しました。これでとりあえず準備ができたので、文法の話をしましょう。

CSSの文法入門

CSSは通常HTMLとは別のファイルで書かれるのでした。 それはレイアウトなどをWebページの基本的な構造の記述とは分離つまり独立して編集できれば実用上良いっていうメリットがあるからでしたね。なので僕はintroduction_to_website_development_jp3.cssを作ったのでした。
CSSではまず最初になにをするかというと、HTMLのどの要素にCSSを適用するかを書きます。 たとえばこのページにおけるh1というタグで書かれているのは具体的には、
              
                <h1>横田健太のWebサイト開発入門 第3講</h1>
と今現在存在しています。これを何かしらレイアウトなどのデザイン変更の対象として、CSSのコードを書きます。
              
                h1 {
                  font-family: "serif";
                  font-size: 36px;
                  font-weight: bold;
                  text-align: center;
                  color: #8B0000;
                }
まずこんな感じで書いてみました。これから説明していきますね。
第1行目ではh1と書かれてその後に「{」という日本語で言えば中かっこでしょうか大かっこでしょうか、なんていうのかよく知らないですがそれがありますね。 この行では、早速HTMLの何の要素のレイアウトなどデザインをこれから設定していくのかというこで、 そういう行為の対象となるHTMLのある要素を1つ決めるのです。この部分をCSSではセレクターと言います。
第2行目はフォントファミリーの指定をしているのですが、ざっくり言えばフォントファミリーとはHTMLの文字の種類を指定するようなプロパティが書かれています。じゃあプロパティってなんやねんっていうと、 要素のスタイルを指定するものです。だから、font-family: "フォント名 " って書くとフォントが指定できるのです。僕はserifにしてみました。なんでって?ググって出てきたものからテキトーに選んだだけで意味なんてないですよ（笑）。 第3行目では何をしているのかはもう簡単で、フォントの大きさを指定しています。
第4行目はフォントを太字にしています。
第5行目では文字列を中央に寄せています。WordやPowerPointなんか使っている人ならすぐわかりますね、 そういうことだってCSSではできるのです。
第6行目では文字列の色を指定しています。colorというプロパティを使っています。 このプロパティではあなたの好きなように文字の色を決めることができます。僕は暗い赤が好きでそうしてみました。 文字の色を決める(指定する)方法は色々とあるらしいですが、ググったら大抵見つかります。例えばすごく簡単な例を出すと、 
              
                color: blue;
と書けば青色になりますし、緑にしたかったら、
              
                color: green;
と書けばよいです。でも他にもやりかたがあり僕は直接色の名前を表す英単語をcolorプロパティでは使っていませんね、 HEXという方法を使いました。HEXというのは16進数で色を表現する方法です。 緑色にしたかったらさっきのように色の英単語ではなくHEXで次のようになります。
              
                color: #00ff00;
といった感じです。詳しくは今ここでは触れません。
第7行目でこのセレクターの終わりを示してこれで終わりです。セレクターの終わりにもかっこは必要なので、かっこで必ず閉じてください。
そしてかっこのかたまりを宣言ブロックと言います。というかどうでもよいかもしれませんが、宣言ブロックの中身のプロパティの終わりにはセミコロンを使っていて、C言語を思い出しましたね。
ここまでの内容を図解すればこんな感じです。
